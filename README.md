# Bachelor Logging

Dette repoet inneholder daglige arbeidslogger for FlowCRT bachelorprosjektet.

## Struktur

- `logs/` - Daglige arbeidslogger organisert etter dato (YYYY-MM-DD.md)

## Formål

Loggene dokumenterer den daglige fremdriften på FlowCRT-prosjektet, inkludert:
- Kodeendringer
- Tekniske beslutninger
- Utfordringer og løsninger
- Neste steg

Loggene skrives på norsk (bokmål) og fungerer som en bachelorprosjektdagbok.
